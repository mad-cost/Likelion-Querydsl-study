## 📝Querydsl 
QueryDSL이란 SQL, JPQL 등을 코드로 작성할 수 있도록 해주는 빌더 오픈소스 프레임워크이다.

- Querydsl을 사용해야 하는 이유
  1. Querydsl은 Java코드를 기반으로 동작한다
  2. Java코드로 동작하기 때문에 컴파일 단계에서 오류를 발견 할 수 있다
  3. 동적 쿼리를 만드는데 유용하다
